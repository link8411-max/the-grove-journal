## Phase 2: 콘텐츠 확장 (마케팅 우선순위 높음)

### 2-1. 봇 주제 확장
현재 6개 → 최소 20개로 확장. 카테고리별 균형 맞추기.

**추가 주제 예시:**
- medical-column: 골반 비대칭, 일자목 vs 거북목, 족저근막염, 어깨 충돌증후군, 손목터널증후군(테크노밸리 직장인), 틀어진 골반과 생리통
- power-pilates: 리포머 vs 캐딜락 차이, 조셉 필라테스 생애, 클래식 vs 모던 필라테스
- grove-story: 회원 변화 사례(익명), 원장님 미국 연수기, 계절별 몸 관리법

**구현**: `marketing_bot.py`의 `TOPICS` 리스트에 추가하면 됨.

### 2-2. 발행 스케줄 최적화
현재: 매일 1편 (같은 주제가 반복될 수 있음)
변경안: **주 3편** (월-의학칼럼 / 수-파워필라테스 / 금-그로브소식)

**구현**: `daily-publish.yml`의 cron을 요일별로 분리하고, 봇에 카테고리 필터 추가.

### 2-3. 네이버 블로그 티저 자동화
매거진에 전문이 올라가면, 네이버 블로그에 **앞부분 300자 + "전문 읽기" 링크**를 자동 게시.
네이버 검색 유입 → 매거진 유입 → 체류시간 증가 → 브랜드 권위.

**구현**: 네이버 블로그 API 또는 Selenium 기반 자동 포스팅 스크립트.

## Phase 3: SEO 강화

### 3-1. Schema.org 구조화 데이터
- `MedicalBusiness` 스키마 (사업체 정보)
- `Article` 스키마 (각 아티클)
- `Person` 스키마 (원장님 전문가 프로필)

**효과**: 구글 검색 리치 스니펫 노출. "광교 필라테스 재활" 검색 시 별점/전문가 정보 표시.

### 3-2. 구글 서치 콘솔 등록
- sitemap.xml 제출
- 색인 요청
- 검색 성과 모니터링

### 3-3. 네이버 서치어드바이저 등록
- 네이버 웹마스터 도구에 사이트 등록
- 네이버 검색 노출 최적화

### 3-4. 아티클별 SEO 키워드 타겟팅
봇이 글을 생성할 때 frontmatter에 `keywords` 필드 추가.
`<meta name="keywords">` + `Article` 스키마의 `keywords` 속성에 반영.

## Phase 4: 전환율 최적화 (CRO)

### 4-1. 아티클 내 자연스러운 CTA 삽입
봇 프롬프트에 "본문 중간에 자연스럽게 재활 접근의 중요성을 강조하되, 영업 멘트 없이" 지시 추가.
독자가 "나도 전문가한테 봐야겠다"고 느끼게 유도.

### 4-2. 카카오톡 상담 채널
네이버 플레이스 예약 외에 카카오톡 채널 추가.
MobileReserveBar에 카카오 상담 버튼 병렬 배치.

### 4-3. 방문자 통계
- Vercel Analytics (무료 티어) 또는 Google Analytics 4
- 어떤 아티클이 예약 전환에 기여하는지 추적
- UTM 파라미터로 네이버 블로그 → 매거진 유입 측정

## Phase 5: 비주얼 업그레이드

### 5-1. 실제 스튜디오 사진
Unsplash 이미지 → 원장님이 직접 촬영한 스튜디오/기구/수업 사진으로 교체.
`public/images/` 또는 외부 이미지 호스팅(Cloudinary) 활용.

**이것이 브랜딩에 가장 큰 차이를 만듦.** 스톡 사진은 고급스러워 보이지만 신뢰감은 실제 사진이 압도적.

### 5-2. 원장님 프로필 페이지
`/about` 페이지 신설:
- 국민대 석사 학위
- Power Pilates 뉴욕 본원 인증
- GRATZ 공인 스튜디오
- 10년 임상 경력
- 자격증/수료증 이미지

### 5-3. OG 이미지 개선
현재: 텍스트 기반 → 카테고리별 배경색 + 로고 + 제목 조합으로 업그레이드.
SNS 공유 시 클릭률(CTR) 직접 영향.

## Phase 6: 장기 확장

### 6-1. 이메일 뉴스레터 (Resend + React Email)
충분한 콘텐츠가 쌓인 후 (20편+), 이메일 구독 기능 도입.
Resend API는 무료 티어 월 3,000건 → 초기에 충분.
홈 ConnectCTA에 이메일 입력 폼 복원.

### 6-2. 다국어 지원 (영문)
광교 외국인 거주자 타겟. 봇에 영문 번역 프롬프트 추가.
`/en/medical-column/...` 경로 또는 언어 토글.

### 6-3. 회원 후기 / 비포-애프터
동의를 받은 회원의 변화 사례를 `grove-story` 카테고리에 게재.
**가장 강력한 전환 콘텐츠.** 단, 의료법 주의 (진단/치료 표현 금지).

### 6-4. 커스텀 도메인
`thegrovejournal.com` 또는 `journal.pilatesgrove.com` 등
Vercel에서 커스텀 도메인 연결 가능 (무료).
